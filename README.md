# Credit Risk Prediction System

## Overview

This project builds a machine learning model to predict the likelihood of a loan applicant defaulting. The goal is to support financial institutions in making data-driven lending decisions and reducing financial risk.

## Problem Statement

Loan default is a major challenge in the financial sector. Incorrect risk assessment can lead to significant financial losses. This project aims to classify applicants into high-risk and low-risk categories.

## Dataset

The dataset used in this project contains applicant financial and demographic information such as income, loan amount, employment status, and credit history.

(Source: Kaggle Credit Risk Dataset)

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training (Random Forest)
5. Model Evaluation
6. Model Explainability (SHAP)

## Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* SHAP
* Streamlit (for deployment)

## Results

The model successfully identifies high-risk applicants using classification techniques. Evaluation metrics such as precision, recall, and confusion matrix were used to assess performance.

## Business Impact

This system can help financial institutions:

* Reduce loan default rates
* Improve decision-making
* Minimize financial losses
